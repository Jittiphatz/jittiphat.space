## Mori Music

เป็นบอทดิสคอร์ดเปิดเพลงที่เขียนด้วยภาษา TypeScript มีคำสั่งและมีฟีเจอร์ที่น่าสนใจมากมาย เช่น การค้นหาเพลงจาก YouTube, Spotify, SoundCloud และอื่นๆ รวมถึงการสร้างเพลย์ลิสต์และการจัดการเพลงในเซิร์ฟเวอร์ดิสคอร์ด สามารถสร้างห้อง requset เพลงได้

## Features

- **รองรับการเล่นจาก**: Youtube , Spotify, SoundCloud, Deezer, MixCloud, และอื่นๆ
- ใช้ **Lavalink เวอร์ชั่น 4.0.8** | Plugins : LavaSrc , Skybot Lavalink , LavaSearch
- UI หน้าเล่นเพลงออกแบบมาอย่างสวยงามและสะดวก
- สามารถสร้างห้อง requset เพลงได้
- สามารถสร้างเพลย์ลิสต์ได้
- สามารถใส่ฟิลเตอร์ระหว่างเล่นเพลงได้

## Library

- **MoriLink**: Lavalink client library for Discord.JS adapted from RainLink.
- **Discord.JS**: A powerful library for interacting with the Discord API.
