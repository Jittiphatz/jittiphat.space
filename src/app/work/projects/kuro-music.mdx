## Kuro Music

เป็นบอทดิสคอร์ดเปิดเพลงที่เขียนด้วยภาษา JavaScript มีคำสั่งและมีฟีเจอร์ที่น่าสนใจมากมาย เช่น การค้นหาเพลงจาก YouTube, Spotify, SoundCloud และอื่นๆ

## Features

- **รองรับการเล่นจาก**: Youtube , Spotify, SoundCloud, Deezer, MixCloud, และอื่นๆ
- ใช้ **Lavalink เวอร์ชั่น 4.0.8** | Plugins : LavaSrc , Skybot Lavalink , LavaSearch , LavaLyrics , Youtube-source
- UI หน้าเล่นเพลงออกแบบมาอย่างสวยงามและสะดวก
- สามารถใส่ฟิลเตอร์ระหว่างเล่นเพลงได้

## Library

- **RainLink**: Lavalink client library for Discord.JS
- **Discord.JS**: A powerful library for interacting with the Discord API.
